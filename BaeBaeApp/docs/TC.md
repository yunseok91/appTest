# BaeBae App — Test Case 문서

## TC-01. 거래 등록

| ID | 테스트 항목 | 입력 | 기대 결과 | Pass/Fail |
|----|------------|------|-----------|-----------|
| 01-01 | 지출 정상 등록 | 카테고리: 식비, 금액: 5000, 메모: "점심", 시간: 점심 | 홈 화면 내역에 표시됨 | |

| 01-08 | 19개 카테고리 전부 노출 | 카테고리 선택 화면 진입 | 19개 항목 모두 보임 | |
| 01-09 | 현금/카드 선택 | 결제수단 전환 | 선택한 수단이 내역에 표시 | |

## TC-02. 거래 내역 화면

| ID | 테스트 항목 | 입력 | 기대 결과 | Pass/Fail |
|----|------------|------|-----------|-----------|
| 02-01 | 전체 내역 표시 | 내역 탭 진입 | 등록한 전체 거래 표시 | |
| 02-02 | 지출 필터 | "지출" 탭 선택 | 지출 항목만 표시 | |
| 02-03 | 수입 필터 | "수입" 탭 선택 | 수입 항목만 표시 | |
| 02-04 | 월별 이동 | < > 버튼 클릭 | 해당 월 내역으로 전환 | |
| 02-05 | 내역 상세 팝업 | 거래 항목 클릭 | 하단 팝업 정상 표시 | |
| 02-06 | 수정 버튼 (내 거래) | 내 거래 상세 → 수정 | 수정 화면 진입 | |
| 02-07 | 삭제 버튼 (내 거래) | 내 거래 상세 → 삭제 | 삭제 확인 알림 → 삭제 후 목록에서 제거 | |
| 02-08 | 파트너 거래 잠금 표시 | 파트너 거래 상세 진입 | 수정/삭제 버튼 없고 잠금 안내 표시 | |
| 02-09 | 팝업 닫기 | 팝업 바깥 영역 탭 | 팝업 닫힘 | |
| 02-10 | 팝업 시스템바 겹침 없음 | 팝업 하단 확인 | 버튼이 시스템 네비게이션바 위에 표시 | |

## TC-04. 댓글 기능

| ID | 테스트 항목 | 입력 | 기대 결과 | Pass/Fail |
